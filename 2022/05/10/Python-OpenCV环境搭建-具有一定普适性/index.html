二.OpenCV安装

1.移动至Anaconda

在Tensorflow的配置博客中,我们介绍了Anaconda的基本情况.此时也需要使用Anaconda进行安装:

2.pip Install

接下来,使用Windows的CMD,通过cd命令进入到放置opencv包的文件夹下,然后pip install xxx(xxx是包的名字)

比如对于我的机器来说,指令就是:

pip install opencv_python-4.4.0-cp36-cp36m-win_amd64.whl

如果有报错信:息的话注意看一下是不是python的环境变量没有设置好或者是没有开启虚拟环境

如果还是报错:

则可能的原因是下载的包的版本和python的版本不一样,具体在上述网站中按照python的版本进行下载.

参考链接:
